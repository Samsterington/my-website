What will you learn?

The main programming language that I will teach you is JavaScript. It is one of the most powerful, and certainly most useful programming languages around today as JavaScript is the key functional language of the internet. When teaching coding, most instructors will start with HTML and then CSS (two other language also used in websites) before they teach JavaScript. Maybe this is because it seems easier to teach the less powerful, non dynamic languages first; however, I do this the other way around.

I have built a course that lets us delve straight into the real code without it being daunting or difficult. The code that lets us do amazing things from the get go. The code that is entertaining to learn and exciting to master. Have a look in the Your Work! section to see some examples of what my amazing students have created.

To teach Javascript in an accessible and entertaining way I use a JavaScript library called p5.js. If that doesn't mean anything to you don't worry, it simply allows us to quickly and easily draw shapes which we can then manipulate and animate using our own code. p5.js also has a free, online code editing software which will also allow us to save all our projects online so that it can be accessed anywhere we want. I encourage students to practice coding between sessions, exploring and creating their own ideas, and this p5.js editing software makes it really easy to do that. I also give you access to my course account so that you can look at any of my code at any time.

Click on the p5 logo on the left to have a look at the p5.js editing software that we'll be coding on in our sessions. You can even have a go with it right now if you're feeling confident; or, if you would like to looks at some example code click on file in the top left hand corner, then select examples and choose one to have a look at. I recommend 'Hello P5 : animate' for some very simple coding, and 'Hello P5 : flocking' for something a bit more complex.

After we've mastered the art of JavaScript, we can then use our skills to develop and build our own websites and online applications. To do this we'll learn how to use HTML, a mark-up language that allows us to create headings, paragraphs, buttons and all the thing you might see on a website. With our JavaScript already up to scratch, HTML becomes a powerful tool to help present our new coding skills.

We'll also delve into the world of CSS, another markup language (like HTML) which will allow us to style the elements of our websites and generally make them look slick, beautiful and all manner of positive adjectives. At this point you will be ready to delve into any area of programming that you want to, so you can pick where we go. Creating games, designing apps or just blasting through your maths homework.
